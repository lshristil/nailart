Tipz & Toez is a creative nail art inspired landing page built using HTML and CSS. I created this project to practice front-end development while designing something visually aesthetic.
The website focuses on a clean layout, stylish background elements, responsive design, and interactive hover effects to create a modern user experience.
One of my favorite parts of this project was experimenting with colors, images, and overall UI styling to match the nail art theme.

Features:
* Responsive landing page
* Modern and aesthetic UI design
* Interactive hover effects
* Custom background styling
* Nail art image showcase
* Social media icon section
* Clean and organized code structure

Tech Stack used:
* HTML5
* CSS3
* Google Fonts
* Font Awesome

While building this project, I improved my understanding of:
* Responsive web design
* CSS layouts and positioning
* Hover effects and transitions
* UI/UX design basics
* Structuring front-end projects
* Using Git and GitHub for project management
